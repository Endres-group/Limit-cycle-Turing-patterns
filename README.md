# How nature discovers rare Turing islands: exploration by common limit cycles
### This repository contains Jupyter notebooks to simulate the paper.

***

### Organisation:
Each notebook contains the code to simulate the data, and generate visualisations from them. The code is annotated to guide the user, as a tutorial.
Each notebook (except Supplementary_figure_dimensionality.ipynb) is expected to run independently without the requirement of running other notebooks. Thereby, certain functions will be repeated. 
- `Figure_2.jpynb` - SEF visualisation. Figure 2 data simulation and visualisation.
- `Figure_3.jpynb` - Discovery of Turing islands. Figure 3 data simulation and visualisation, alongside SFig 6,7,13
- `Figure_4.jpynb` - Reproducibility of Turing patterns. Figure 4 data simulation and visualisation, alongside SFig 8
- `Figure_5.jpynb` - Enhanced reproducibility through French flag gradients. Figure 5 data simulation and visualisation, alongside SFig 12
- `Supplementary_figure_velocity.jpynb` - Effect of varying angular velocity of limit cycles to Turing pattern formation. SFig 2
- `Supplementary_figure_SEF_properties.jpynb` - Analysis of SEF variation under different pattern setups. SFig 3,4
- `Supplementary_figure_dimensionality.jpynb` - Effect of dimensionality on limit cycle coupled Turing patterns. SFig 9,10
- `Supplementary_figure_multistability.jpynb` - Effect of multistability within the system. SFig 11
Requirements:
- Python 3.9 or higher, 8 CPUs.
